checkbox需要水平排列
只需要在label标签上添加类名“checkbox-inline”
radio需要水平排列
只需要在label标签上添加类名“radio-inline”

下拉菜单

  1. 在使用Bootstrap框架的下拉菜单时，必须调用Bootstrap框架提供的bootstrap.js文件。
  2. 在使用Bootstrap框架中的下拉菜单组件时，其结构运用的正确与否非常的重要，如果结构和类名未使用正确，直接影响组件是否能正常运用。
  3. 使用一个名为“dropdown”的容器包裹了整个下拉菜单元素，
  4. 使用了一个<button>按钮做为父菜单，并且定义类名“dropdown-toggle”和自定义“data-toggle”属性，且值必须和最外容器类名一致，
  5. 下拉菜单项使用一个ul列表，并且定义一个类名为“dropdown-menu”
  6. 下拉菜单对齐方式：pull-right、dropdown-menu-right、dropdown-menu-left。同时一定要为.dropdown添加float:left css样式
